# Monthly Business Review Report (Streamlit)

Tables-first dashboard + PDF export (no charts).

## Deploy (click-only)
1) Create a Public GitHub repo
2) Add `app.py`, `requirements.txt`, `README.md`
3) Streamlit Cloud: https://share.streamlit.io → New app
4) Select repo → branch `main` → file `app.py` → Deploy

## Usage
- Upload Shipment details
- Upload optional RCA sheets
- Set filters → click "Run report"
- Export PDF
