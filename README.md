# Monthly Business Review Report (Streamlit)

Upload shipment execution data + RCA sheets, explore the dashboard, and export a designed A4 landscape PDF report.

## Deploy (click-only)
1. Create a public GitHub repo
2. Add `app.py`, `requirements.txt`, `README.md`
3. Go to Streamlit Community Cloud: https://share.streamlit.io
4. New app → pick your repo → branch `main` → file `app.py` → Deploy

## Updating
Edit files on GitHub → Commit to `main` → Streamlit auto redeploys.
